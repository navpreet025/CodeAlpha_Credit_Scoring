# CodeAlpha_Credit_Scoring
# Credit Scoring Model

## Project Overview
This project builds a machine learning model to predict credit risk based on customer financial information.

## Objective
To classify customers into different credit risk categories using machine learning algorithms.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

## Machine Learning Models
- Logistic Regression
- Random Forest Classifier

## Steps Performed
- Data Loading
- Data Cleaning
- Missing Value Handling
- Categorical Feature Encoding
- Model Training
- Model Evaluation

## Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1-score
- ROC-AUC Score

## Result
Random Forest achieved excellent classification performance on the dataset.
